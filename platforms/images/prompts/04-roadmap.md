# 适配路线图提示词

## 用途
展示 Phase 1 和 Phase 2 的时间线，放在文章"关键时间节点"和"两阶段适配策略"章节。

## 文章核心信息
| 阶段 | 时间 | 核心任务 |
|------|------|---------|
| Phase 1 | **2026-04-28 前** | SDK 构建适配：keyWindow、SceneDelegate、StoreKit 2、Privacy Manifest、UIDesignRequiresCompatibility |
| Phase 2 | **Xcode 27 发布前 (~2026-09)** | Liquid Glass 完整适配：移除兼容标志、safeArea 调整、背景色协调 |

## 推荐尺寸
1920 x 1080（16:9 横版）

---

## Gemini / ChatGPT / Claude 提示词

**视觉目标**：从左到右"两步走"，左边紧急（橙色/红色），右边规划（蓝色）。时间线上的节点要对应文章中的具体任务。

```
生成一张两阶段流程图，横版 16:9，深色背景接近纯黑。

【核心 / 发光时间线，贯穿全图】
画面中央偏下，有一条从左到右的水平粗线条，占画面高度约 8%。线条是渐变色：左端亮橙色（#FF6B00），向右平滑过渡到亮蓝色（#007AFF），右端深蓝色。线条有外发光，像霓虹灯管。这是全图视觉引导线，必须一眼看到。

【第一阶段卡片 / 左侧】
时间线左端上方，有一个大圆角矩形卡片（占画面宽度 24%、高度 30%）。
- 卡片顶部有一条橙色发光边框
- 卡片内最显眼位置：一个白色的扳手/工具图标，很大（占卡片高度 40%）
- 图标下方有 3 个极小的图标排成一排（像工具栏）：齿轮（设置）、锁（隐私）、购物车（StoreKit）
- 这 3 个小图标暗示 Phase 1 的具体任务
- 卡片下方有一个橙色发光圆点连接在时间线上
- 卡片整体有向上浮起感（底部有弥散阴影）
- 卡片左上角有一个极小的日历页角标，暗示"4月28日"

【第二阶段卡片 / 右侧】
时间线右端上方，有一个同样大小的圆角矩形卡片。
- 卡片顶部有一条蓝色发光边框
- 卡片内最显眼位置：一个白色的画笔/设计图标，很大（占卡片高度 40%）
- 图标下方有 2 个极小的图标：一个水滴/玻璃球（Liquid Glass）、一个矩形框（safeArea）
- 这 2 个小图标暗示 Phase 2 的具体任务
- 卡片下方有一个蓝色发光圆点连接在时间线上
- 同样有浮起感

【时间线中间】
两个阶段之间，时间线上有一个小节点，表示过渡点。节点是一个小型 Xcode 图标（锤子形状），暗示"Xcode 27 发布"这个关键转折点。

【背景】
纯黑背景（#050505）。时间线下方有非常淡的水平网格线（像坐标纸），透明度不超过 8%。画面最左侧有极淡的橙色光晕，最右侧有极淡的蓝色光晕，面积很小。

【绝对不要】
不要真实文字、不要日期数字、不要箭头符号。用颜色（橙→蓝）和位置（左→右）表达时间先后。用图标表达具体任务，而不是文字。

风格：极简信息图，苹果 Keynote 发布会风格，高级感。像苹果 WWDC 幻灯片中的一页，每个元素都在讲故事。
```

## 如果两个阶段区别不够明显

```
第一阶段卡片的背景用暖色调（深橙红），第二阶段卡片的背景用冷色调（深蓝）。两个卡片的背景色差异要足够大，让人一眼区分。时间线的渐变也要更明显，中间过渡区域拉长。
```
