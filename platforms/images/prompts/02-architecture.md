# 项目架构概览图提示词

## 用途
介绍项目四大核心能力，放在文章"自动化扫描工具"和"模板速览"章节附近。

## 文章核心信息
项目提供 4 大能力：
1. 🔍 扫描 — `ios26-scanner.py`，20+ 规则检测废弃 API
2. 📋 模板 — Swift/OC 双语言代码模板，直接复制使用
3. 📚 文档 — FAQ、测试指南、检查清单
4. 🔄 更新 — 持续迭代，v1.5.0 已覆盖 15+ 盲点

## 推荐尺寸
1600 x 900（16:9 横版）

---

## Gemini / ChatGPT / Claude 提示词

**视觉目标**：一眼看出 4 个方块，扫描和模板稍微突出（因为文章重点讲这两个）。

```
生成一张信息图，横版 16:9，深色背景接近纯黑。

【主体 / 4 个等大方块】
画面正中央，均匀排列 4 个圆角矩形方块，横向一排，每个方块约占画面宽度的 16%、高度的 38%。4 个方块处于画面黄金分割位置（偏上）。

方块 1（最左，扫描）：
- 深橙色背景
- 中央有一个巨大的白色放大镜图标，占方块高度的 50%
- 放大镜镜片内有极小的代码符号 `{ }`，暗示扫描代码
- 这是文章重点能力之一

方块 2（左中，模板）：
- 深蓝色背景
- 中央有一个巨大的白色文档图标（像一张纸，右上角有折角），占方块高度的 50%
- 文档图标上有两条横线，暗示代码行
- 这也是文章重点能力之一

方块 3（右中，文档）：
- 深绿色背景
- 中央有一个巨大的白色书本/手册图标（像一本打开的书），占方块高度的 50%
- 书页上有极小的横线，暗示文字内容

方块 4（最右，更新）：
- 深紫色背景
- 中央有一个巨大的白色循环箭头图标（两个箭头组成圆形），占方块高度的 50%
- 箭头旁有极小的版本号"v1.5"的感觉

【连接关系】
4 个方块之间有极细的白色半透明线条连接，线条带微弱发光。线条很细（不超过 2 像素），只是暗示关系，不能比方块抢眼。

【强调扫描和模板】
扫描方块（最左）和模板方块（左中）的边框可以稍微亮一点（白色细边框发光），比其他两个方块更醒目一点点，因为文章重点讲这两个能力。

【背景】
纯黑背景（#0A0A0A）。方块下方有非常淡的倒影，像放在黑色玻璃桌面上，倒影透明度不超过 15%。背景中没有任何其他元素。

【绝对不要】
不要文字、不要标题、不要说明标签、不要数字编号、不要多余装饰。只留 4 个彩色方块 + 白图标 + 极细连接线 + 淡倒影。

风格：极简、苹果设计奖级质感、高对比度、每一个像素都有目的。像苹果官网的组件展示图。
```

---

## 如果 4 个方块太平淡

```
给扫描方块（橙色）和模板方块（蓝色）加一层极淡的外发光，让它们从背景中微微浮现。其他两个方块保持平面。这样视觉重点自然落在扫描和模板上。
```
