# 封面图提示词

## 用途
文章顶部主封面，用于掘金/CSDN/公众号等平台。

## 文章核心信息
- iOS 26 SDK 强制适配，**4 月 28 日** deadline
- 提供自动化扫描脚本 + 代码模板
- 开源项目，可直接用于生产

## 推荐尺寸
1920 x 1080（16:9 横版）

---

## Gemini / ChatGPT / Claude 提示词

**视觉目标**：第一眼看到"iOS 26"，第二眼看到"deadline 紧迫"，第三眼看到"工具/解决方案"。

```
生成一张科技博客封面图，横版 16:9，深色背景。

【画面最核心 / 占 C 位】
画面上半部分中央，有一个巨大的"iOS 26"标题组合：
- "iOS" 用白色、较细的无衬线字体，位于左上方
- "26" 用极粗的巨大数字，占据画面上半部约 30% 高度，颜色是电光蓝到青色的渐变发光，带有玻璃折射质感
- "26" 是整个画面最亮、最大、最抓眼的元素

【第二焦点 / 传达紧迫感】
"iOS 26" 的右下方，有一个醒目的日历/倒计时元素：
- 一个圆角矩形卡片，像日历页
- 卡片内有一个红色的日期标记（像 deadline 标记）
- 卡片旁边有一个小型的时钟/沙漏图标，暗示时间紧迫
- 这个元素用红色/橙色强调，与蓝色"26"形成对比

【第三焦点 / 解决方案】
画面下半部分，有一台 MacBook Pro，屏幕显示 Xcode 代码界面，代码中有黄色警告标记和修复后的绿色对勾。电脑屏幕发出柔和的白光。电脑大小约占画面宽度的 35%，位于画面底部中央偏左。

【背景 / 极简不抢戏】
极深的蓝黑色背景（#080C14），背景中有非常淡的、像数字雨一样垂直下落的细小光点，暗示"代码/数据"。背景亮度必须远低于前景。

【点缀 / 极小、半透明】
- 画面左下角：Python 图标（蛇形），半透明，暗示自动化脚本
- 画面右下角：Swift 图标（鸟类），半透明

【绝对不要】
不要中文文字、不要段落文字、不要多余装饰。画面要极度干净，只有"iOS 26" + deadline 标记 + MacBook + 深色底 + 两个极小图标。

风格：苹果发布会级简洁，深色模式，高对比度，电影级光影。传达"紧迫但有解决方案"的情绪。
```

---

## 如果 deadline 元素不够明显

```
把日历/倒计时卡片放大到"26"数字的 1/3 大小，放在"26"的右下角，用亮红色发光边框强调。这是画面第二重要的元素。
```
