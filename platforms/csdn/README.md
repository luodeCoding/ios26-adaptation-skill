## 发布步骤

1. 打开对应 `.md` 文件
2. 全选复制（Cmd+A / Ctrl+A）
3. 打开 CSDN 编辑器，选择 **Markdown 编辑器**
4. 粘贴内容
5. 设置专栏分类：**移动开发 > iOS**
6. 添加标签：`iOS 26`、`Xcode 26`、`Swift`、`Objective-C`、`移动开发`
7. 文章类型：**原创**
8. 点击发布文章
