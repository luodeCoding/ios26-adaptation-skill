## 发布步骤

1. 打开对应 `.md` 文件
2. 全选复制（Cmd+A / Ctrl+A）
3. 打开掘金编辑器，切换到 **Markdown 模式**
4. 粘贴内容
5. 上传封面图（建议尺寸：1200x600）
6. 添加标签：`iOS`、`iOS 26`、`Swift`、`Objective-C`、`Xcode`
7. 点击发布
